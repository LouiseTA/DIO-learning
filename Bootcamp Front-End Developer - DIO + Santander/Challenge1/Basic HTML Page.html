O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação (não é linguagem de programação) utilizada para estruturar o conteúdo de páginas web.

Estrutura básica de um documento HTML


            <!DOCTYPE html>
            <html lang="pt-br">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Título da página>
            <head>
            <body> </body>
            </html>
        
        
    
Principais tags aprendidas


    
Estrutura e texto

    
    
        
  • <h1> a <h6> - Títulos (h1 é o maior, h6 o menor)
    • 
        
  • <p> - Parágrafos
    • 
        
  • <br> - Quebra de linha (não precisa de fechamento)
    • 
        
  • <hr> - Linha horizontal
    • 
        
  • <strong> - Negrito (ênfase forte)
    • 
        
  • <em> - Itálico
    • 
    


    
Links

    
Usamos a tag <a> para criar links:

        
<a href="https://www.dio.me" target="_blank">Visite a DIO</a>

    
    
        
  • href: define o destino do link
    • 
        
  • target="_blank": abre o link em uma nova aba
    • 
    


    
Imagens

        
<img src="imagem.jpg" alt="Descrição da imagem" />

    
    
        
  • src: caminho da imagem
    • 
        
  • alt: texto alternativo, útil para acessibilidade
    • 
    


    
Listas

    
Ordenada:

        

        <ol>
            <li>Primeiro item</li>
            <li>Segundo item</li>
        </ol>


    
Não ordenada:

        

        <ul>
            <li>Item A</li>
            <li>Item B</li>
        </ul>


    
Tabelas

        

        <table>
            <tr>
                <th>Nome</th>
                <th>Idade</th>
            </tr>
            <tr>
                <td>Ana</td>
                <td>28</td>
            </tr>
        </table>

    
    
        
  • <tr>: linha
    • 
        
  • <th>: cabeçalho
    • 
        
  • <td>: célula de dados
    • 
    


    
Divisões e seções

    
    
        
  • <div>: divisão genérica (usada com CSS)
    • 
        
  • <section>: agrupa uma seção de conteúdo
    • 
        
  • <header>: cabeçalho da página ou seção
    • 
        
  • <footer>: rodapé
    • 
        
  • <article>: conteúdo independente (como uma notícia)
